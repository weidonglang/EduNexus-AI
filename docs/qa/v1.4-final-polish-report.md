## 新增功能列表

- AI 调用日志搜索与等级字段。
- AI 聊天会话、模型切换和历史消息。
- 学籍异动附件管理员材料弹窗、预览、下载和中文类型显示。
- 压测面板后端 API 课程刷新模式。
- 批量任务中心。
- 数据归档清理页面和记录导出。
- 数据库只读查询模板。
- 演示环境健康巡检脚本。

## 不做功能说明

以下功能故意排除在 v1.4 范围外，避免范围失控和高风险数据操作：

- 智能备课与教学资源库。
- 课程资源文件夹、视频资源库、课程克隆、资源推送。
- 数据库备份、定时备份、恢复、备份下载。

## 健康巡检结果

巡检脚本：

```powershell
.\scripts\health-check.ps1
```

本地 IDEA 后端运行在 8080 时可使用：

```powershell
.\scripts\health-check.ps1 -BackendUrl http://localhost:8080
```

输出：

```text
reports/health-check-YYYYMMDD-HHmmss.md
reports/health-check-YYYYMMDD-HHmmss.json
```

检查项包括后端、前端、MySQL、Redis、Nacos、上传目录、Docker Compose 文件和 release zip。

## 已知限制

- 数据归档清理在演示环境不做物理删除，只记录任务和审计，避免误删演示数据。
- 数据库浏览器只允许白名单 SELECT 模板和表预览，不开放任意写 SQL。
- AI 服务离线时继续使用本地 fallback，日志等级会体现 WARN/ERROR。
