教学综合信息服务平台 · API 接口总览

后端接口清单与答辩说明

Vue 本身不自带后端接口文档。本页面是根据 Spring Boot Controller 的 @RequestMapping、@GetMapping、@PostMapping、 @PutMapping、@DeleteMapping 整理出的本地静态接口总览。

答辩口径： 前端 Vue 通过 Axios 调用这些后端接口，后端统一返回 ApiResponse。 管理员接口大多在 /api/admin/** 下，学生个人接口多用 /api/students/me/**， 教师端接口集中在 /api/teacher/**。

认证与菜单

登录、退出、当前用户、动态菜单。

5 个
方法接口说明
POST/api/auth/login账号密码登录，返回 token 和用户信息。
POST/api/auth/logout退出登录，清理当前会话。
GET/api/me查询当前登录用户信息。
GET/api/menus根据当前角色返回可见菜单。
GET/api/dashboard/me当前用户首页仪表盘数据。

个人信息、学籍异动、报名类申请和材料上传。

方法接口说明
